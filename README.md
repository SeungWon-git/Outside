### * 영상에는 포함되지 않았지만 구현된 기능
  1. 최대 4인 멀티 플레이 지원
  2. 출혈 시스템 
    * 좀비에게 피격 당할 시 랜덤 확률로 출혈 발생 → 출혈 발생 시 체력 서서히 저하 & 속도 + 공격력 감소
    * 지혈 아이템을 사용해야 출혈을 멈출 수 있음. 하지만 지혈 또한 랜덤 확률로 성공 여부가 나뉨
  3. 아이템 떨구기 기능 → 동기화 작동
